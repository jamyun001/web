# 모바일 사용법

1. Termux 설치 ( 앱스토어 )

2. pkg install nodejs git

3. npm install ws uuid

4. git clone https://github.com/jamyun001/web

5. cd web

6. cd [ 파일 ]

7. unzip [파일].zip

8. rm -rf [파일].zip

9. start.bat -> start.sh

10. chmod +x start.sh ( 권한 부여 )

11. ./start.sh ( 실행 )

12. 마크 인게임 /connect localhost:3000
