# 모바일 사용법

1. Termux 설치 ( 앱스토어 )

2. pkg install nodejs

3. pkg install git

4. npm install ws uuid

5. git clone https://github.com/jamyun001/web

6. cd web

7. cd [ 파일 ]

8. unzip [파일].zip

9. rm -rf [파일].zip

10. start.bat -> start.sh

11. chmod +x start.sh ( 권한 부여 )

12. ./start.sh ( 실행 )

13. 마크 인게임 /connect localhost:3000
