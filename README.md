# 모바일 사용법

1. Termux 설치 ( 앱스토어 )

2. pkg install nodejs

3. npm install ws uuid

4. cd web

5. cd [ 파일 ]

6. unzip [파일].zip

7. rm -rf [파일].zip

8. start.bat -> start.sh

9. chmod +x start.sh ( 권한 부여 )

10. ./start.sh ( 실행 )

11. 마크 인게임 /connect localhost:3000

## 웹소켓을 사용해서 얻은 불이득은 책임지지 않습니다.
